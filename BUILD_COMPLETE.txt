QUICK START
═══════════════════════════════════════════════════════════════════════════

1. Install Dependencies:
   pip install -r requirements.txt

2. Configure Environment:
   cp .env.template .env
   # Add ANTHROPIC_API_KEY to .env

3. Run System:
   python main.py

4. Access Services:
   • Web UI: http://localhost:8501
   • API: http://localhost:8000
   • API Docs: http://localhost:8000/docs

FEATURE HIGHLIGHTS
═══════════════════════════════════════════════════════════════════════════

✅ True Agentic Architecture
   - 4 independent agents with clear responsibilities
   - No direct agent-to-agent coupling
   - All communication through MCP servers & orchestrator

✅ Tool Use Pattern
   - Agents use Claude SDK with tool_use
   - MCP servers expose focused functions as tools
   - Transparent for evaluation

✅ LangGraph Orchestration
   - StateGraph-based workflow
   - Sequential execution with clear control flow
   - State management at each step

✅ Explainable Decisions
   - Every decision includes reasoning
   - Factors ranked by importance
   - Complete audit trail

✅ Production Ready
   - Type safety (Pydantic)
